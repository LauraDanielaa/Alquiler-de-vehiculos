# Sistema de Gestión de Alquiler de Vehículos 

Este es un repositorio destinado para el proyecto académico desarrollado para la materia de Bases de Datos Avanzadas de la Universidad Distrital Francisco José de Caldas.

El proyecto consiste en el desarrollo de un sistema de información orientado a la gestión de alquiler de vehículos. La aplicación busca facilitar la administración de clientes, vehículos, reservas, alquileres y pagos mediante una base de datos relacional conectada a una API REST. El sistema está enfocado en optimizar la organización de la información y garantizar procesos más eficientes dentro de una empresa de alquiler de automóviles.

A lo largo del proyecto se implementan diferentes conceptos vistos en la asignatura, incluyendo diseño relacional, procedimientos almacenados, triggers, vistas, índices, subconsultas y transacciones en PostgreSQL. Además, se incorporan mecanismos que permiten validar información, mantener la integridad de los datos y registrar auditorías sobre las operaciones realizadas en el sistema.

El repositorio contiene los scripts SQL necesarios para la creación y configuración de la base de datos, así como el código fuente de la API y la documentación relacionada con el desarrollo del proyecto. También incluye el modelo entidad-relación, los requerimientos funcionales implementados y ejemplos de las funcionalidades principales del sistema.

## Tecnologías utilizadas

- PostgreSQL
- PL/pgSQL
- API REST
- Git y GitHub

##  Contenido del repositorio

- Scripts SQL de creación de tablas y relaciones.
- Procedimientos almacenados y triggers.
- Vistas, índices y consultas avanzadas.
- Código fuente de la API REST.
- Modelo entidad-relación del sistema.
- Documentación del proyecto.

##  Funcionalidades generales

El sistema permite gestionar información relacionada con clientes, vehículos, reservas y alquileres. También incluye procesos de validación de disponibilidad de vehículos, registro de pagos, control de mantenimientos y almacenamiento de auditorías para garantizar la trazabilidad de las operaciones realizadas.
