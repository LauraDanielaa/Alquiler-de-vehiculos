Índices

Se añadieron tres índices:

1. Índice sobre estado de alquiler (idx_alquiler_estado)

Optimiza búsquedas de alquileres según su estado activo, finalizado y cancelado

2. Índice sobre estado de vehículo (idx_vehiculo_estado)

Mejora el rendimiento de consultas relacionadas con la disponibilidad de vehículos.

3. Índice sobre fechas de reserva (idx_reserva_fechas)

Reduce el tiempo de búsqueda de conflictos entre fechas.
