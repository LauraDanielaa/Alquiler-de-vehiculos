Registrar Pago (registrar_pago)

Permite registrar pagos asociados a un alquiler de forma segura y atómica.

Funcionalidades
Verifica que el alquiler exista.
Comprueba que el alquiler se encuentre activo.
Calcula el total pendiente del alquiler.
Valida que el monto ingresado no exceda el saldo pendiente.
Registra el pago en la tabla pago.
Revierte toda la operación si ocurre algún error.
Beneficios
Evita pagos duplicados o superiores al valor adeudado.
Mantiene la consistencia financiera del sistema.
