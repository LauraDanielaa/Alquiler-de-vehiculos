Vistas
 
Se realizaron dos vistas para la base de datos 

1. Vista de detalle de alquileres (vista_detalle_alquileres)

Centraliza información relacionada con un alquiler.

Incluye
Datos del cliente.
Información del vehículo.
Categoría del vehículo.
Sucursal asociada.
Total de pagos realizados.

Uso: Facilita consultas completas sin necesidad de múltiples JOIN.

2. Vista de vehículos disponibles (vista_vehiculos_disponibles)

Muestra únicamente los vehículos disponibles para alquiler.

Incluye
Información del vehículo.
Categoría.
Datos de la sucursal.
Uso: Optimiza consultas de disponibilidad para la aplicación.
